# 📝 Summarizer-HF – Text Summarizer Web App

**Summarizer-HF** is a simple yet powerful text summarization web application.  
It uses **HuggingFace Transformers (T5 model)** with **FastAPI backend** and a clean **HTML frontend** to generate concise summaries of long text.

---

## ✨ Features

- **Transformer-based Summarization**
  - Powered by HuggingFace `T5ForConditionalGeneration`
  - Supports long text inputs (up to 512 tokens)
  - Generates concise summaries with beam search

- **Frontend UI**
  - Clean and responsive HTML interface
  - Paste or type text directly
  - One-click summarization

- **Backend API**
  - FastAPI endpoints for summarization
  - JSON response with summary
  - Easy integration with other apps
